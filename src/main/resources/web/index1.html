Our research is distributed machine learning
We design and implement a distributed machine learning system called DRPS

DRPS is a distributed machine learning system based on parameter server with a disk-resident schema. We propose a partitioning algorithm on parameter models called PPMCD and a parallel control model called WSP to improve performance
